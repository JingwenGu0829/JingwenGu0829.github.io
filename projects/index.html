MoltFocus

Runtime Infra Application

MoltFocus is an agent-native daily planner that co-evolves with you through a built-in scheduling engine that generates time-blocked plans while learning from your nightly check-ins and reflections. The system features constraint-based task scheduling, pattern analytics that adapt recommendations over time, dual web and terminal interfaces, a 22-endpoint REST API, and file-based storage (YAML, JSON, Markdown) that keeps you in control without vendor lock-in.
